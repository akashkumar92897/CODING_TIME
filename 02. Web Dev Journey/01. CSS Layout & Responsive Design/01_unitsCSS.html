CSS Units Playground

Open DevTools and experiment with values. Resize the browser window to understand viewport units.

1. px (Pixels)

Fixed size. Doesn't depend on parent or viewport.

width: 300px
height: 80px

2. em

Parent font-size: 20px
font-size: 2em (40px)
padding: 1em (40px)

3. rem

Root font-size is currently: html { font-size: 16px; }

font-size: 2rem (32px)
padding: 1rem (16px)

4. Percentage (%)

Parent Width = 500px
width: 50%
padding: 5%

5. vw and vh

width: 50vw
height: 10vh

Resize the browser window.

6. vmin

Size depends on the smaller viewport dimension.

7. vmax

Size depends on the larger viewport dimension.

8. ch

Width is based on the width of the character "0".

9. lh

This container has line-height: 2.
height: 3lh

Try changing the line-height value and observe the green box.

Exercises

  1. Change html font-size from 16px to 20px.
  2. Change parent font-size in the em section to 30px.
  3. Resize browser width and observe vw.
  4. Resize browser height and observe vh.
  5. Switch between portrait and landscape window shapes and compare vmin/vmax.
  6. Change input width from 30ch to 50ch.
  7. Change line-height from 2 to 3 and observe 3lh.
  8. Create a card using only rem units.
  9. Create a square using only vmin.
  10. Create a hero heading using vmax.
